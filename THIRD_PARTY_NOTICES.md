# Avisos de terceros

Luna Fetch utiliza software de terceros. Este documento no sustituye los textos completos de sus licencias.

- **Kotlin y Kotlinx** — Apache License 2.0.
- **Compose Multiplatform** — Apache License 2.0.
- **Ktor** — Apache License 2.0.
- **Coil** — Apache License 2.0.
- **yt-dlp** — The Unlicense; los binarios pueden incorporar dependencias adicionales.
- **FFmpeg** — LGPL o GPL según las opciones de compilación y bibliotecas incluidas.
- **youtubedl-android** — GPL-3.0; utilizado por el target Android para integrar Python, `yt-dlp` y FFmpeg.

El código propio del repositorio conserva su licencia MIT. Cada APK distribuido se publica bajo los términos aplicables de GPL-3.0 por incluir `youtubedl-android`; el código fuente correspondiente está disponible en el tag de cada release y este aviso acompaña al repositorio.
